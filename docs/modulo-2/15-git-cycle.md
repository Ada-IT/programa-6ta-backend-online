### 📝 Contenidos

**Operatoria de git**

- Clonar repositorio en local
- Configurar usuario local
- Subir cambios
- Realizar commit
- Crear ramas nuevas a partir de master
- Generar pull request
- Unificar el codigo con merge
- Solucionar conflictos
- Compartir repositorio a otros usuarios
